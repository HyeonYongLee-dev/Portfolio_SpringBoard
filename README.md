# 스프링 프레임워크를 이용한 게시판 포트폴리오 - 개인 프로젝트
## 목적
<p> 1. 신입 개발자의 기본 소양 배양.</p>
<p> 2. 기본적인 crud 기능에 여러 기능을 조합한 게시판을 제작.</p>
<p> 3. 반복적인 연습을 통해 개발 요구사항에 최적화 될 수 있는 개발속도를 배양.</p>
